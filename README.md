Welcome to my personal corner of the web! This repository contains the source code for my portfolio and blog, where I document my journey in **Cybersecurity and Information Assurance**.

## 🎯 About This Project
This site serves as a central hub for my professional profile, technical projects, and blog posts regarding network security, penetration testing, and digital forensics.

* **Portfolio:** Highlighting my academic background in Computer Science and my specialized path in Cybersecurity and Information Assurance.
* **Blog:** Sharing insights on certification prep (CySA+, PenTest+, CISM) and security best practices.
* **Tech Stack:** Built with Astro, Tailwind CSS v4, and TypeScript for a type-safe, performant experience.

## ✨ Features
* **Performance:** Ultra-fast static site generation (SSG) via Astro.
* **Design:** Fully responsive with a native dark mode system for long nights of research.
* **Security Focus:** A clean, professional layout designed to showcase certifications (ISC², CompTIA) and technical write-ups.
* **Content:** Dynamic blog powered by Markdown & MDX with estimated reading times.
* **SEO:** Fully optimized with OpenGraph tags, sitemaps, and RSS feeds.

## 🛠️ Technical Stack
| Component | Technology |
| :--- | :--- |
| **Framework** | Astro |
| **Styling** | Tailwind CSS v4 |
| **Logic** | TypeScript |
| **Deployment** | GitHub Pages / Netlify |

## 📁 Project Structure
* `src/content/blog/`: Markdown/MDX files for blog posts.
* `src/data/`: Structured data for certifications, skills, and work history.
* `src/components/ui/`: Reusable interface elements (Timeline, Badges, etc.).

## 🚀 Quick Start
To run this project locally for development:

1. **Clone the repository:**
   git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
   



2. **Install dependencies:**
npm install




3. **Start the dev server:**
npm run dev
   Visit `http://localhost:4321` to see it in action.
